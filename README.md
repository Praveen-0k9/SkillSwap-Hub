# SkillSwap Hub – A Student Collaboration Platform

SkillSwap Hub is a web-based platform designed to help students connect, collaborate, and share skills with each other. The system allows users to create profiles, list their skills, search for other students based on skills, and communicate through a messaging system.

The main objective of this project is to promote peer-to-peer learning and collaboration by providing a simple and efficient platform where students can exchange knowledge and work together on different tasks.

## Features

* User Registration and Login
* Profile Management
* Skill Listing and Search
* Real-time Chat System
* User-friendly Interface

## Technologies Used

* Frontend: HTML, CSS, JavaScript, React.js
* Backend: Node.js, Express.js
* Database: MongoDB

## Purpose

This project is developed as part of the Bachelor of Computer Applications (BCA) curriculum to demonstrate practical implementation of web development and database management concepts.
